第三步：讓 Gemini 幫你寫出具備「偵錯」功能的網頁

⚠️ 真實錯誤案例：當額度用完時

✗ 失敗：You exceeded your current quota, please check your plan and billing details. For more information on this error... Quota exceeded for metric: generativelanguage.googleapis.com/generate_content_free_tier_requests... Please retry in 23.78s.

💡 白話文解釋：

「這代表你問得太快了，或是今天的免費次數已經用光了。請休息 23 秒後再試一次！」

請複製以下提示詞並貼給 Gemini，它會幫你把生硬的錯誤訊息變成上面的白話文：

請幫我生成一個單一 HTML 檔案，包含以下功能：
1. 介面頂部有 API Key 輸入框，以及一個下拉選單供選擇對話風格。
2. 使用 JavaScript 串接最新的 Gemini API 模型。
3. **核心功能：錯誤白話文回饋**。如果 API 請求失敗（例如報錯 "Quota exceeded"），請將生硬的英文錯誤翻譯成白話文告訴使用者。例如：「問太快了，請稍等再試」或「API Key 貼錯了」。
4. 介面要現代化，使用 Tailwind CSS，對話要有氣泡感，並在等待回覆時顯示「思考中...」的動畫。
5. 所有程式碼（HTML/CSS/JS）需在同一個檔案內。

📋 部署清單

確認檔名是否為全小寫 index.html (檔名錯了網頁會打不開)。

進入 Settings > Pages，Source 選擇 "Deploy from a branch"。

Branch 選擇 main，資料夾選擇 / (root)，點擊 Save。

✨ 預期成果

發佈成功後，你會得到一個網址，這代表你已經將 AI 助手部署到了全球雲端：

https://你的帳號.github.io/專案名/

💡 提示：部署通常需要 1~2 分鐘才會生效，請耐心等待。
