🤖 API 互動教學教室

從申請 API 到 GitHub 雲端部署，完成你的第一個 AI 專案

第一步：取得 API Key

1

訪問 Google AI Studio 並登入。

2

點擊左側 "Get API key" 並創建一個新的 Key。

3

複製這串密碼，這就是你程式碼中 `apiKey` 變數要填入的值。

💡 提示： API Key 就像銀行密碼，請不要貼給不認識的人。
🔑

"API Key 是與 AI 溝通的通行證"

API 是如何對話的？

💻

➡️ Request (請求)

你的網頁把「問題」和「API Key」打包成 JSON 包裹送出去。

⬅️ Response (回應)

Gemini 算出答案後，把文字裝進包裹回傳給你的網頁。

最後一步：把網頁發佈到網路上

想把這個聊天機器人傳給朋友玩嗎？使用免費的 GitHub Pages 服務：

1

準備好檔案

將你的程式碼存檔，檔名一定要叫 index.html。

2

上傳至 GitHub

在 GitHub 點擊 New Repository，取個名字（例如 my-ai-bot），然後將檔案拖進去上傳。

3

開啟網頁開關

進入該專案的 Settings > Pages，在 Branch 選擇 Main 並按下 Save。

4

獲得網址！

等待約 1 分鐘，你會得到一個網址：
https://你的帳號.github.io/my-ai-bot/
🚀
挑戰： 試著在 GitHub 網頁上輸入你剛剛申請的 API Key，看看對話功能是否運作正常！
